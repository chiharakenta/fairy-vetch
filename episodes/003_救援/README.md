# 救援
すっかり日も暮れて、あたりは暗くなり始めていた。
リュックのジッパーを閉じて背負い直し、あたりを見回す。

「帰り道、どっちだったっけ…」

来た時は草をかけ分けたはずだが、すぐに戻ってしまったのか、耕介が通ったはずの道は跡形もなくなっていた。

（まあ、歩いてれば見つかるか。）

気を取り直し、そのあたりの草をかき分けて入ろうとするが、

「いてて！」

来た道には無かったはずの棘のある草に腕を突かれ、たまらず引き返す。

（もしかして、俺やばい？）

ここに来て初めて、耕介は勢いのまま入ったことを後悔した。
来た時は枝で足元を叩きながら進んだが、帰り道を切り開くほどの力はない。


（どっちだ？）

太陽がどの方向から照らしていたか必死に思い出そうとする。
しかし、ベッチに気を取られていたので、それももはや定かではない。

唯一の手がかりは、夕日が沈んだ山の方角。
だが、耕介の住む町は2方向から山に囲まれている。
正しい道か、逆方向かは2つに1つ。
危険な賭けに出るしかない。

「ベッチ、ねえ起きてよ。」

リュックの中で眠っているであろう友達に声をかける。



...しかし、カエルの鳴き声が草むらの外から聞こえるだけだった。
思った以上に深い眠りについているらしい。


ようやく耕介は自分の状況を理解した。
理解してしまった。

「え、え、どうしよ、どうしよ。」

冷や汗が額を伝う。
汗ばんだTシャツが体温を奪い、体が震えだす。
だが、異変はそれだけではなかった。

（かゆい）

腕を見てみると、あちこち腫れている。

「かゆい...」
思わず口をついてつぶやいた。

「かゆい...！」
腕をかき始めると、今度は足までかゆみが広がってきた。

「かゆい！！！」
三回目は...もはや悲鳴だった。



（なんで俺、気づかなかったんだろ。）

耕介は、知っているはずだった。
兄がいつも、伸縮性のある長袖インナーの上にシャツを着ているのを見ていた。

近所の森林組合のおじさんも、地の厚い作業服を着て仕事に出ていた。
理由を尋ねた時は、怖い話口調で「蚊がうじゃ〜ってくるんだぞおぉぉ」と喋る顔に、思わず笑った記憶もある。


しかし、忘れていた。
ベッチの声に引っ張られ、全部忘れていた。
それも、自分にとって一番必要な時に。


学校のテストでは、テスト時間に思い出せなければ間違いになる。

"半袖半ズボンで草むらに入ったらどうなるのか"

...この問題の答え合わせとして、腕と足には、たくさんの赤い間違いマークをつけられてしまった。


最後の頼みの綱として、スマートフォンに手を伸ばす。

（兄ちゃん…！）

だが、バッテリーの残量は残り5％。
日頃からゲームで消耗してきたツケが回ってきた。

（早く！早く！！）

急いで操作をしようとするが、顔認証が通らない。
慌ててパスコードで認証しようとするも...


数字を打ち終わると、振動だけが響き入力がリセットされた。
パスコードを打ち間違えたらしい。
手に嫌な汗が滲む。

（もう！）

もう一度手早く打ち直す。
しかし、汗で滑ってしまったのか、またも振動が響くだけだった。


（くそ！）

焦りが怒りに変わる。
指を叩きつけるように画面の数字を打つと...


「あ」


"連続で間違えたためロックされました。5分後に再度お試しください"

本来であれば、持ち主を守るためのメッセージが表示される。
そして、画面右上のバッテリー表示は残り3％。

5分後も使えるのかどうか
...保証は全くなかった。



スマホを握る指から力が抜ける。
周りの景色が、プールに入ったようにぼやけてきた。



...その時だった



「おーい！！」


−−−声が聞こえた。
はっきりと。
間違いなく。



「耕介！居るんだろ！！」

ーーーまた声が聞こえた。
おそらく、
道の方から。


耕介は、その声をよく知っていた。



「にい…ちゃん。」

大きく息を吸い込むと、体中から熱いものが込み上げてきた。
全力でそれを吐き出すと、

「うわーん！にいぢゃ〜ん！たずげで〜！！」

数日前、お気に入りのゲームでゲームオーバーになった時とは、比べ物にならない声が出た。
視界はもはや、流れる水に遮られて何も見えなかった。


「そこか！」

不意に目の前が真っ白になって目を瞑る。

「はは、ハハハ！」

すると、先ほどとは打って変わって兄の笑い声が響き渡る。

「観念しろ、畠山耕介(はたけやまこうすけ)！お前は完全に包囲されている！」

眩しさが止み、目を開けると、懐中電灯の光が近くの草を照らしていた。
どうやら、見つけてもらえたらしい。

「兄ちゃん！」

全身の力が抜け、胸の内にあった重りがどっと溶ける。
本人の姿は確認できないが、こんな状況で刑事ドラマのものまねをする人物を、耕介は一人しか知らなかった。

わざとらしく鼻声を効かせた耕平が言葉を続ける。

「お前は決して破ってはならない法を破ったのだ。お前の家族が泣いているぞ！」

そこまで言うと鼻声は消える。

「"ほんとに大丈夫か"ってな…！」

最後の一言だけは、真剣な声だった。

スマホのメッセージを思い出し、耕介はたまらず叫ぶ。

「ほんとごめ～ん！！」

耕平はそれに答えることはなく、

「今行くからな！」

と一言言うと、サッサッと足音を立てながら一直線に近づいてくる。

「まったく誰だよ、こんな時間にそんなところでお話しする友達なんて…悪い子とは友達になるなって普段から言ってるだろ。」

ぶつぶつと文句を言いながら、順調に声は近づいてきたが

「いてて、この…」

棘のある草に引っかかったらしい。
瞬間、身を案じる耕介だったが、

「邪魔だ！どけ！！」

兄の一声と共に、ザクッと心地よい音を立てて、背の高い影が横に倒れていく。

倒れた影の先には…

片手に鎌を持った兄の姿があった。
耕介とは違い長袖長ズボン。
長靴と作業用のゴム手袋まで完備している。

頭には、たまにしか見ないゴムベルト式のヘッドライトを取り付けていた。
トレードマークのブルーライトカットメガネも、今日ばかりは光って見える。


耕介の頬が緩む。

「兄ちゃ〜ん！」

抱きつこうとした瞬間、

「この大バカやろう！！」

兄の一喝が、耕介の耳をつんざいた。

「門限破ってこんな時間に何やってんだお前は！！
あと鎌持ってんだから、不用意に近づくな！」

耕介には、全くもって言い返す言葉はなかった。

「ごめん、俺、フェアリーベッチの種集めてて。」

耕介の言葉を聞いた耕平は驚く。

「ヘアリーベッチ...？前に話した緑肥（りょくひ）の名前、覚えてたのか。」

微妙に発音の異なる名前を聞いて、耕介は首を傾げる。

「ヘアリーベッチ？」
「あぁ、空気中の窒素を固定してくれて...」

そこまで言うと、耕平は顔をしかめ、自分の耳元を手でぱっぱと払った。

「説明は後だ。さっさと帰るぞ。蚊が鬱陶しくて仕方ない。」

踵を返し、来た道を向き直る耕平。
すぐに後ろを振り返り、顔だけ耕介の方に向けると、

「耕介、歩けるか？」

柔らかく落ち着いた声で聞かれた。

「うん。歩けるよ。足は怪我してない。」
「そうか。それなら良かった。」

それだけ言うと、耕平はしゃがみこんだまま草を刈る。
よく手入れをされているのか、硬い草の茎もギコギコと心地よい音を立てて倒れていく。


耕平の後ろには、耕介の通りやすい道ができていた。
