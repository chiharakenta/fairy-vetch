「先に降りていいぞ、耕介」

兄の耕平に促され、耕介は軽トラックを降りる。
すると、いつも住んでいるわが家の匂いが、耕介を大冒険から現実へと帰してくれた。

二人が住む家は、コンクリートの壁に、瓦屋根がついた二階建ての一軒家だ。
祖父母の代に建てたものらしく、白かったはずのコンクリートの壁は、全体的にくすんできており、ところどころ土汚れが目立つ。
しかし、ヒビはなく、雨漏りもこれまでなかった。祖父母の馴染みの大工の腕が良かった証拠だろう。

ぶるぶると鳴っていたエンジン音が止むと、トタン屋根の車庫から耕平が出てきた。
車庫は家の庭をはさんだ先の、道路と隣接したところに建っている。
後輪駐車では一度道路に出なければならないので、耕平はいつも耕介を先に下ろすのだった。

「いや〜まさか、兄ちゃんが軽トラで迎えに来てくれるなんてね〜」

足がクタクタだったので、耕介は車で帰れたことに上機嫌だった。
耕介の顔を見て、耕平がふっとため息をこぼす。

「お前が"友達と話してる"とか言って、わけのわからんメッセージ送るからだろ。あっちこっち行ったんだぞ〜。近所の吉田さんも知らないっていうもんだから。」

「ごめんて。」

軽く謝った耕介だったが、思ったよりも周りを巻き込んでいたらしい。
吉田さんまで心配させたことを思うと、胸の奥が少しだけ重くなった。

「もしかしたらと思って、農具積んでる軽トラで行ったら、まさかまさかのビンゴだからな。」

「ってことがさあ、あったんだよ！」

フェアリーベッチと出会ってからの顛末を話した。

だが耕平は何も言わず、表情も変えず、ただ口に含んだ肉じゃがをもぐもぐと食べている。

  

妙な静寂が流れる。

20:03頃を刻んでいる時計の音だけが、やたら大きく聞こえていた。

  

ようやくごくんと喉を鳴らすと、耕平は箸を置き一言だけ

「そうか」

と言った。

  

「え？それで終わり？」

耕介は、兄がベッチの存在を信じるとまでは思っていなかったが、小言も何も言われないことが不思議だった。

「もしかして、信じてくれるの？」

淡い期待を持ち始めた耕介だったが、当然そんなことはなかった。

  

「信じるもなにもないだろ。」

そう言って、耕平は深いため息をつくと、椅子から降りてヘナヘナと頭を抱えてしゃがみ込む。

「お前がついに病気になっちまうなんてな…俺の育て方が悪かったのか。」


病気という言葉を聞き、耕介は胸がざわついた。
汗が額をつたる。
ツバを飲み、恐る恐る尋ねてみる。

「え？俺なんか悪い虫に噛まれちゃった？」

瞬間、耕平がゴホゴホと咳き込む。
慌てて口元を抑え、顔を上げると

「そういうことじゃない！心の病気の話をしてるんだ。」
バツが悪そうに口にしながら席に戻ってきた。

心の病気という言葉を聞いて耕介は納得した。
四年一組の教室に入らなくなってからというもの、スクールカウンセラーとは時々話すが、まだ市内にある精神科には行ったことがなかった。

「ああ、そっか。やっぱり一回ちゃんと病院行って診てもらったほうがいいの？」

耕平の眉が内に寄る。
「いや、診察自体は必要なんだろうが、これはそういうことじゃなくて、皮肉ってやつでな…」
「皮肉？」

そう言って、ようやく耕介は兄の真意を悟った。
「あ～、なにそれ！俺がベッチ見えてること病気扱いしたの！？」
そう言うと、耕平の口元が緩む。
「ようやく気づいたのか。相変わらず鈍いやつだな。」
負けじと耕介も反撃する。
「だから言ってるじゃん！ベッチの種があるんだって。」
そう言ってリュックから豆のサヤを取り出すと、土の粉が舞う。
「おいこら！床が汚れるだろ。」
そこからは、売り言葉に買い言葉だった。
「しーらない！俺のこと信じてくれないからそうなるんだよ〜。」
「そんなこと言って、誰がお前のリュック洗うと思ってんだ。」
「自分で洗うもん！」
「ほんとかー？そんなこと言って、後でやっぱりやりたくないとか泣き出すのがいつものオチだろ。」
「やるったらやる！」

そこまで言うと耕平は引き下がった。
「わかったわかった。お前の言い分は認める。認めるから、話をそいつに戻そう。」
そう言って耕平はサヤを指差す。
「確かに、そいつは明らかにマメ科だ。夏枯れして秋に種を蒔く必要があるんなら、ヘアリーベッチの可能性は十分あるだろう。」
「だからその、ヘアリーベッチってなに」
聞いた瞬間、耕平は机に突っ伏す。
「おい！緑肥(りょくひ)と知ってたから集めたんじゃないのかよ！」
草むらからの帰りの兄の言葉を思い出す。
だが、やはり意味は分からなかった。
「りょくひ？なにそれおいしいの？」
「別に美味しくはない。ああー、いや、緑肥の中にも食べられるものはあるんだが、今はそうじゃなくて…」
「？」
その後も、耕平は訳のわからないことをぶつぶつと呟きながら、緑肥なるものについて思案していた。


たまにあることだった。
幼い頃から、兄は周りの大人に天才と呼ばれていたから、耕介はそういうものなのだろうと理解していた。
「兄ちゃんって、大変なんだね。」
心から同情するが、耕平の反応は
「まったく…誰のおかげでそうなってると思ってんだ。」
苦笑いだった。
