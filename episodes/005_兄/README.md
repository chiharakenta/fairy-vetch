# 兄

風呂場で楽しげに歌う弟の声を聞きながら、耕平は、自室の椅子にもたれかかっていた。

―――弟に友達が出来た。

普通であれば、喜ばしい知らせなのだろう。
それが、普通のお友達なのであれば...

（まさか、ついに草を本気で"友達"とまで言い出すとはな）

耕介には、草の生命力が光として見えている。
それ自体は前々から知っていた。
初めは病気を疑ったが、調べるうちに科学的にもあり得ることだと分かった。

（あった）

PCのメモアプリで検索をかけると、過去に書いたメモがヒットした。

```
# 共感覚（シナスタジア）

- ある刺激に対して、普通は関係ないような感覚が引き起こされる

## 例
- 黒い文字に色が見える
- 音に色や形を感じる
- （人の感情を匂いで読み取る）

## 所感
- 耕介の目は病気ではない
- 芸術的才能の一種なんだろう
- 生活に支障が出るならサポートは必要
```

（この程度に考えてたんだがな...）

耕平は深くため息をついた。

光が見える程度なら、「少し特殊な共感覚」程度で説明はつく。
しかし、植物を人格として捉え、会話までしたとなると話は別だ。
ましてや、連絡なく門限を破り、人の土地にまで勝手に入り込んだ。

耕介の前では茶化していたが、なんらかの妄想や病気を疑わざるを得なかった。


（俺のほうが捜索に回るなんてな...因果応報ってやつか。）

本家の叔父の家を抜け出して、山に入って一人で泣いていた中学生の頃が思い出される。


外で暗くなりだし、耕平が空腹と不安で帰りたいのに帰りきれなかった時、吉田力矢が不意に現れたことがあった。
なぜ場所が分かったのかと尋ねる間もなく、

「とりあえず食え！帰るかどうかはそっから決めればいい。」

と言って、奥さんお手製の弁当を二人分持ってきて耕平と一緒に食べた。
大して会話もしなかったが、その時の弁当の味だけは覚えている。

家に帰った後は、「どこをほっつき歩いてた」と叱る叔父に悪態をついていた。
...実際には、力矢に相談したのは叔父だと、叔母から後で教えてもらったが。


叔父に対する反発心は、消えたわけではない。
家もそれほど裕福ではないのに、事故で亡くなった父や、病気で亡くなった母の葬儀にかなりの額をかけた。
棺や花など、目に見えるコストのかかっているものならいざ知らず、死後の世界の名前に費用をかける意味が理解出来なかった。

「そんなことより耕介の学費にでもとっとけよ！貧乏ジジイ！」

そんなことを叫んで、当時は家を飛び出した。
暴言は悪かったと思っているが、今でも自分の理屈自体は間違っているとは思っていない。


しかし...

（探す側って、こんなに大変なんだな。）

今になって、耕平は自分がかけてきた迷惑の重さを実感していた。

当時、自分を見つけたからと言って、叔父にとっての問題が解決したわけではない。
自分たちに起きたことを受け入れられず、何に対しても噛みつく狂犬のような自分を、一体どうすればいいのか悩んだことだろう。


「兄ちゃん、アイス！！」

風呂場から出てきた耕介の声が響き渡る。
食後は腹がいっぱいだからといって、風呂上がりにお預けしたアイスがあった。
ドアを少し開けて返事を返す。

「ちっちゃい冷凍庫に入れてあるから、一本食べていいぞ！！」

「おっけー！」

すると、ガラガラと冷凍庫の引き出しを開ける音がして、バタンと閉まった。
自室のドアを閉めて、自分の世界に戻る。


実際のところ、こちらも問題は山積みだった。
耕介は、一見楽しそうに見えているが、植物を友達と言うほどまで誰かとの繋がりを求めていたのかもしれない。

学校という場が合わないのは仕方ない。
もう時代は変わった。不登校も決して珍しいものではない。

けれど、時々二人で行く隣町の子ども食堂でも、ほとんど耕平と一緒にキッチンでボランティアをしている。
食べる時間になっても、同年代の子どもたちの輪に入るのが苦手なのか、隅っこの二人席で一緒に食べるだけだった。

（俺が、耕介を一人にさせてしまったのが原因だろうか。）

遠縁の親戚である隣町の天(そら)の元で暮らし始めたが、耕平が県外の大学に行っている間は、耕介を置いていってしまった。
天は懐の深い人だが、教会の副牧師として働いていたので、いつでも家に居られるわけではない。
進学時の判断が今になって悔やまれる。


とはいえ、単なる妄想の一言でも片付け難い。
耕平は、今度はAIチャットアプリを開き、会話ログを眺める。

"ヘアリーベッチの可能性は十分あります。
ただし、正確な情報は元の土地の持ち主にお確かめください。"

種やサヤの画像と共に、夏枯れ・秋撒き・土壌改良などの情報を入力すると、やはりAIもヘアリーベッチの可能性を示唆してきた。
マメ科の妖精(フェアリーベッチ)の存在は疑わしいが、耕介が対話した友達とやらの言っていることは、農学的にはデタラメとも言い切れない。

（やっぱり、確かめてみるしかないか。）

AIにまで確かめるよう言われてしまったのだから、耕平は動くしかなかった。
何より、今までゲームぐらいにしかやる気を持たなかった耕介が、農業に興味を持ち始めたのだ。
たとえきっかけが想像上の友達だったとしても、背中を押してやりたい気持ちはあった。

耕平は、スマホの電話帳を開きながら、朝なら電話できるかと思案していたが、すぐにやめた。
叔父の耳も遠のき始めている。電話で何度も聞き返しが発生するより、直接会って話したほうが早いだろう。
全く気は進まなかったが...


重い腰をなんとか上げて、着替えを出し、風呂場に移動する。
道中、耕介がリビングのマットに寝転がって、何やら真剣な表情で動画を見ていた。
流石に覗き見は悪いと思ったが、気になった耕平が耳をすませてみると、

「どうもなのだ！今回は水流式の自動収穫畑を作っていくのだ！作成に必要なブロックは...」

ゲームの農場作成動画だった。
耳馴染みのある機械音声のキャラクターの声がリビングに響く。

（水流ぐらいで収穫できたら、楽なんだけどなぁ。）

目の前に待ち受けている大変な作業を思うと、ゲームのシンプルな世界が羨ましかった。
