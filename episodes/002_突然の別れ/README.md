# 突然の別れ

他人の土地に植物の種を蒔く。

難しい頼みではあったが、
友達は助け合いだと言った手前、耕介も断るわけにはいかなかった。

「分かった。なんとかならないか兄ちゃんに聞いてみるよ。土地の話とか詳しいし。」

「へー、コースケってお兄ちゃん居るの？」

兄弟という概念が通じ、耕介は安堵した。

「そうなんだよ。俺より10個上で、何でもできるんだ。今日の夕飯だって肉じゃが作ってくれて…」

そこまで喋ってから、はっと気がついた耕介は、慌ててリュックにしまい込んでいたスマートフォンを取り出す。
恐る恐る通知欄を覗くと、


"18:00までには帰ってこいよ〜" - 17:50

"不在着信" - 18:02

"不在着信" - 18:04

"ほんとに大丈夫か？" - 18:05

"不在着信" - 18:10


門限はとっくに過ぎていた。
この時ばかりは、耕介もマナーモードにしていたことを後悔した。

"ごめん！今友達と話してるから終わったら帰る！！" - 18:12

送信ボタンを押すとようやく一息ついた。
振り返ると、肩越しにベッチがスマホを見つめていた。

「なにこれ？テレパシー？」

「まあ、そんなもんかな。」

スマホの原理は説明しようがなかったので、耕介は軽く流すことにした。

ベッチも特に驚くこともなく、
「ふ〜ん」
と流していた。

だが、

「ちゃんと心は通じた？」

完全にテレパシーだと思い込んでいるベッチに答えづらいことを聞かれる。

「多分…俺の言いたいことは分かってくれるはず。」

耕介は自信なさげに答えた。
こんなメッセージで、「友達が出来たのか」と兄が喜ぶはずはない。

どう思われるかという心配は頭から振り払い、ベッチの願いに話を戻す。

「ベッチ、とりあえずここらにある種を集めればいいの？」

「うん！お願い。」

ベッチの返事を聞くと、耕介は豆を収穫し始めた。

兄の手伝い以外で作物の収穫をすることは、耕介にとって初めてだった。
ハサミも用意していないので、両手で鞘ごとちぎり取ってはリュックの中に入れていく。

…リュックを兄が洗う時、何を言われるかと頭によぎったが、これも耕介は考えないことにした。


収穫をしながら、ふと疑問に思ったことを尋ねてみる。

「そういえばさ、"今のまんまだと役に立てない"って言ってたけど、どういうこと？」

「そのまんまの意味だよ〜。種を撒いてもらわないと、土のみんなを元気にできない」

「その、元気に出来ないってのは？」

「う〜んとね...」

ベッチはしばらく上を向いて考えていたが、
何か思いついたように、体がピクリと震える。

「ベッチはね、魔法が使えるの！」

「...え？」

頭が理解を拒んだ。

ゲームが好きな耕介だったが、流石に現実世界に魔法が存在しないことぐらいは分かっている。

「魔法って...何？」

「簡単だよ！土のみんなに"お願い"って言ったらね、ベッチの頼みに答えてくれるの！
そしたらみ〜んな元気になるんだよ！！」

説明を聞いてもいまいち分からなかったが、
ベッチには不思議な力があって、仲間の協力が必要だということは理解できた。

「じゃあ、そのために種を撒く必要があるの？」

「そーそー！ベッチがいっぱい増えないと、魔法が使えないからね。」

どうやら、ゲームと同じく、魔法の発動には条件が必要らしい。
それも、ベッチに必要なのは魔法力ではなく、たくさん種を植えること。

「そっか！じゃあ頑張って種集めないとね。」

「そーなの！お願いね〜」

「うん！」

不思議な妖精も、魔法も、
そして友達も、
耕介が欲しかったものが、今目の前にある。

...収穫する手が速くなった。

「すごいすごい！その調子！」

ベッチの応援が背中を押す。

「まかせて！」


ーーー独りではない。
ただそれだけのことが、耕介にとっての喜びだった。


気がつけば近場に見えていた豆の収穫は終わり、リュックの底を埋め尽くす程度には集まった。

「これだけあれば十分かな？」
ベッチを見上げると…


また羽が萎れていた。


「ぜんぜん足らない」


「…ああ、そっか。」

耕介も言われてから気がついた。
リュックの底を埋めた程度では、種はその半分以下。せいぜい撒けるのは家庭菜園レベルの範囲だ。
近所のおじいさんが農業をしていた頃は、ここ一面に作物が並んでいたと兄から聞いたことがある。

文字通り桁が違った。

耕介は眉間にしわを寄せる。

「どうしたらいいかな？」

ベッチはあっけらかんとしていた。

「仲間を集めたらいいんだよ。」

意外な言葉に耕介は驚いた。

「仲間？」

「ベッチの種を他のところから持ってくるんだよ。」

種を持ってくると聞いて、耕介は疑問に思った。

「他のところから持ってきたら、それはベッチじゃなくて別の子なんじゃないの？」

「ううん。ベッチはね。ベッチたちみんなで一つなんだよ。」

みんなで一つ、と聞いて、耕介は小学二年生の頃に歌った歌を思い出した。

"みんなは一人のために、一人はみんなのために"

友達は、助け合うことで一つになれる。

ベッチの言葉が同じ意味なのかは分からないが、
"みんなで一つ"という言葉には、胸が暖かくなった。

「そっか、みんなお友達なんだね。」

「そうだよ！コースケも、土のみんなもオトモダチ！！」

「分かった！それじゃ、兄ちゃんにも頼んで種集めてみるね。」

「ありがとう！」




気がつくと、いよいよ日も沈もうとしていた。
2つの山の谷間から、赤い夕日が耕介とベッチを照らし出す。
すると、ベッチの薄紫の球体が夕日の色に変化していく。

「あれ？ベッチ、どうしたの？」

ベッチは答える。

「そろそろね。おやすみの季節なんだ。」

よく見ると、夕日の色に変わっているのではない。
ベッチの姿が薄くなっていた。

「寝るってこと？」
「そうだよ。次の秋までね。」

その言葉を聞いて耕介も理解した。
今、ベッチの本体である豆をすべて収穫し終えたばかりなのだ。

植物は種を残すと、次に芽吹くまで"休眠期間"がある。

それは耕介にとって、
出会ったばかりの友達と、しばらく会えなくなることを意味していた。


「そんな、せっかく会えたのに。」

「大丈夫だよ！」

答えるベッチの輪郭が段々とぼやけていく。

「種、撒いてくれるんでしょ？」

「それは...そうだけど。」

こんなにすぐに別れるなど、耕介は想像もしていなかった。
上手く挨拶の言葉が出てこない。

「ごめん、そろそろホントに眠くなってきちゃった。」

ベッチの体が完全に消えかかっている。

「ベッチ...」

名前を呟くことしかできずにいると

「大丈夫。また、会えるよ。」

そう言って、別れの挨拶のように羽を左右に振ると、ベッチの体が細かい緑の光の粒子に分解されていく。

「…おやすみ」

ベッチは最後にそう言い残すと、耕介のリュックの中に吸い込まれて消えていった。

慌ててリュックをおろして中を開けるとそこには、



…ただ収穫し終わった豆があるだけだった。



「…おやすみ。ベッチ」
別れを惜しむ間もなく消えていった友達の姿を見て、耕介はようやく挨拶を返した。

豆が無くなった草むらに、夕日だけが差し込む。
明日からここに来ても...ベッチはいない。


再び出会うために、やるべきことは一つだけ。

「俺、農業、やってみるよ。」

自分の決意を口にした。
眠っている友達に、聞こえていると信じて。
